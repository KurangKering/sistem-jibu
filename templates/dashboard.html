Selamat Datang di Sistem Normalisasi Bahasa Makassar

Sistem ini merupakan implementasi dari penelitian yang berjudul "Normalisasi Bahasa Makassar". Ada beberapa menu yang terdapat pada sistem ini, antara lain:

Kamus
Menu kamus merupakan menu yang menampilkan seluruh data kamus. data kamus digunakan untuk proses preprocessing hingga normalisasi. Terdapat proses tambah data kamus secara manual atau dapat juga melakukan import data secara massal menggunakan file berekstensi .xlsx dengan format column dimulai dari nomor;kata;arti kata. Selanjutnya data kamus dapat diubah ataupun dihapus.
Data Uji
Pada menu ini pengguna dapat melihat data uji, menambah data uji secara manual atau melalui import data berekstensi .xlsx dengan format column nomor;data uji, merubah data uji, serta menghapus data. Seluruh data uji yang dimasukkan akan digunakan untuk proses normalisasi.
Cleaning
Menu berikutnya adalah cleaning. Apabila pengguna mengakses menu ini, maka sistem akan melakukan proses preprocessing terhadap seluruh data uji dan menampilkan hasil preprocessing kedalam bentuk table.
Normalisasi Data Uji
Menu pengujian data uji menampilkan seluruh data uji dan pada setiap data uji terdapat satu tombol normalisasi. Ketika pengguna menekan tombol normalisasi, maka sistem akan melakukan proses normalisasi. Hasil normalisasi akan ditampilkan sistem sesaat setelah sistem selesai melakukan proses normalisasi.
